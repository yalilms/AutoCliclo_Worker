# AutoCiclo Mobile

Esta es una app móvil que hice para gestionar piezas y vehículos de desguace. Está hecha con React Native, Expo y usa SQLite para guardar los datos.

## Cómo usar

1. Instala las dependencias con `npm install`
2. Ejecuta en Android con `npx expo run:android` (o en iOS si tienes Mac)
3. Necesitas tener Node.js y un emulador o dispositivo físico

## Qué hace

- Puedes añadir, editar y borrar piezas y vehículos
- Hay un inventario para ver qué piezas tiene cada coche
- Se pueden tomar fotos y escanear códigos QR
- Muestra los vehículos en un mapa

Solo funciona compilando nativo, no va con Expo Go.
